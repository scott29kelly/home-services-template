**Steps:**

1. Run `npm run build` to generate fresh production build with all optimizations applied
2. Start preview server: `npx vite preview` in background (port 4173)
3. Wait for server to be ready

4. For each of the 5 page template groups, run Lighthouse 3 times desktop + 3 times mobile:

Pages to measure:
- Homepage: `http://localhost:4173/`
- Service page: `http://localhost:4173/roofing`
- Static page: `http://localhost:4173/about`
- City page: `http://localhost:4173/service-areas/anytown`
- Blog post: `http://localhost:4173/resources/best-roofing-materials-harsh-winters`

Desktop command (3 runs each):
```bash
npx lighthouse {URL} --preset=desktop --output json --output-path ./lh-tmp.json --quiet --chrome-flags="--headless=new --disable-dev-shm-usage"
```

Mobile command (3 runs each):
```bash
npx lighthouse {URL} --output json --output-path ./lh-tmp.json --quiet --chrome-flags="--headless=new --disable-dev-shm-usage"
```

Extract scores:
```bash
node -e "const r=require('./lh-tmp.json'); const c=r.categories; const a=r.audits; console.log(JSON.stringify({perf:Math.round(c.performance.score*100), lcp:(a['largest-contentful-paint'].numericValue/1000).toFixed(2), tbt:Math.round(a['total-blocking-time'].numericValue), cls:a['cumulative-layout-shift'].numericValue.toFixed(3)}))"
```
