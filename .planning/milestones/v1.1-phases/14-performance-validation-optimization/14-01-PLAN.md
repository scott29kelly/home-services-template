**Part A: Build pipeline — vite.config.ts**

In the `copyAndOptimizeImages()` plugin, after the existing optimization logic (after writing the optimized/copied image), add responsive variant generation for hero images.

Add this logic inside the `imageFiles.map` callback, after the existing try/catch block that writes the optimized image:

```typescript
// Generate responsive variants for hero images
const heroPattern = /hero|Hero/
if (heroPattern.test(relPath) && ext === '.webp') {
  const srcForResize = optimized && optimized.byteLength < originalSize ? optimized : srcBuffer

  // 768w variant (mobile)
  const w768 = await sharp(srcForResize).resize(768).webp({ quality: 80 }).toBuffer()
  const w768Path = path.join(destDir, relPath.replace('.webp', '-768w.webp'))
  await fs.mkdir(path.dirname(w768Path), { recursive: true })
  await fs.writeFile(w768Path, w768)
  results.push(`  ${relPath} [768w]: ${(w768.byteLength / 1024).toFixed(0)}kB`)

  // 1280w variant (tablet/mid-size)
  const w1280 = await sharp(srcForResize).resize(1280).webp({ quality: 80 }).toBuffer()
  const w1280Path = path.join(destDir, relPath.replace('.webp', '-1280w.webp'))
  await fs.writeFile(w1280Path, w1280)
  results.push(`  ${relPath} [1280w]: ${(w1280.byteLength / 1024).toFixed(0)}kB`)
}
```

This matches all hero images: `hero-roofing.webp`, `siding-hero.webp`, `about-hero.webp`, `contact-hero.webp`, `storm-damage-hero.webp`, `projects-hero.webp`, `service-areas-hero.webp`, `testimonials-hero.webp`.

**Part B: Hero component — src/components/sections/Hero.tsx**

Add a helper function to build srcset strings:

```typescript
/** Build responsive srcSet for WebP hero images with 768w, 1280w, and full-size variants */
function buildHeroSrcSet(src: string): string | undefined {
  if (!src.endsWith('.webp')) return undefined
  const base = src.replace('.webp', '')
  return `${base}-768w.webp 768w, ${base}-1280w.webp 1280w, ${src} 1920w`
}
```

Update the `<img>` element to include srcSet and sizes:

```tsx
<img
  src={backgroundImage}
  srcSet={buildHeroSrcSet(backgroundImage)}
  sizes="100vw"
  alt=""
  role="presentation"
  fetchPriority="high"
  width={1920}
  height={1080}
  className="absolute inset-0 w-full h-full object-cover"
/>
```
