**Step 3 — Create `src/lib/blog.ts`:**
Build the blog engine module following the pattern from 04-RESEARCH.md Pattern 1:

- Use `import.meta.glob('../content/blog/*.md', { query: '?raw', import: 'default', eager: true })` to import all markdown files as raw strings. Type the result as `Record<string, string>`.
- If TypeScript complains about the glob type, add a `src/types/markdown.d.ts` declaring `declare module '*.md?raw' { const content: string; export default content; }`.
- Import `matter` from `gray-matter` to parse frontmatter.
- Define a Zod schema for post frontmatter with these fields:
  - `title`: z.string().min(1)
  - `slug`: z.string().min(1).regex(/^[a-z0-9-]+$/)
  - `date`: z.string().regex(/^\d{4}-\d{2}-\d{2}$/)
  - `excerpt`: z.string().min(1)
  - `tags`: z.array(z.string()).min(1)
  - `coverImage`: z.string().min(1)
  - `author`: z.string().default('Acme Home Services')
  - `published`: z.boolean().default(true)
- Export `PostFrontmatter` type inferred from the schema.
- Export `BlogPost` interface extending PostFrontmatter with a `content: string` field (raw markdown body).
- Export `getAllPosts()`: iterate over glob modules, parse each with gray-matter, validate frontmatter with safeParse, skip invalid or unpublished posts (warn to console), check for duplicate slugs (warn and skip), sort by date descending (newest first). Cache the result in a module-level variable so repeated calls do not re-sort.
- Export `getPostBySlug(slug: string): BlogPost | undefined`.
- Export `getAllTags(): string[]` — deduplicated, sorted alphabetically.
- Export `getPostsByTag(tag: string): BlogPost[]`.

- "All" pill is active by default (shows all posts).

- Active tag pill should be visually distinct (filled background in brand-blue, white text). Inactive pills: outline/ghost style.
- When a tag is clicked, reset pagination to page 1.

- Use `useState` for page number (client-side pagination is fine since all posts are loaded eagerly).
- If total posts <= POSTS_PER_PAGE, hide pagination entirely.

**Empty state:**
